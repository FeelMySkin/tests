Библиотека для работы с программным I2C: soft_i2c.c и soft_i2c.h

В целом, данную библиотеку можно использовать на любой архитектуре, главное - передать в неё функции работы с ножками, и вызывать функцию SoftI2C_Tick с частотой 2*(Нужная частота I2C).

Сделал пример программы для контроллера STM32F401CC.
